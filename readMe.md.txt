# Weather-Journer app Project


## Overview
This project requires you to create an asynchronous web app that uses web API and
user data to dynamically update the UI


## Instructions
This will require modifying the `server.js` file and the `website/app.js` file. you can see `index.html` for element
references, and once you are finished with the project steps, you can use `style.css` to style your application to customized perfection.



## Extras
If you are Interested in testing your code as you go, you can use `test.js as a template for writing and running some basic test for your code.
